Installation

Step 1: Clone the repository

git clone https://github.com/yukihamada/groq-go.git && cd groq-go

Step 2: Build

make build

Step 3: Set your API key

export GROQ_API_KEY="your-api-key"

Step 4: Run CLI

./bin/groq-go

Or: Run Web Interface

./bin/groq-go -web
